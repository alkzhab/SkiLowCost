# SkiLowCost

**SkiLowCost** est un **site web** qui permet de **réserver facilement un séjour de ski**.  
Il propose une interface pour les clients pour réserver leur séjour, ainsi qu’un **dashboard pour le gestionnaire** afin de suivre et gérer toutes les réservations.

## Fonctionnalités

### Pour le client
- **Réservation de séjour** : Choisis les dates, la station et le type de séjour.  
- **Paiement sécurisé** : Finalise la réservation en toute sécurité.  
- **Confirmation et historique** : Reçois un récapitulatif et consulte tes réservations passées.  

### Pour le gestionnaire
- **Dashboard complet** : Visualise toutes les réservations, triées par date, client ou type de séjour.  
- **Gestion des réservations** : Modifie, valide ou annule les réservations directement depuis le tableau de bord.  
- **Statistiques** : Suivi des réservations et tendances pour optimiser la gestion.  

## Technologies utilisées
- **Frontend** : HTML, CSS, JavaScript  
- **Backend** : PHP
- **Base de données** : PostgreSQL
